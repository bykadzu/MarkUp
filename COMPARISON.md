# MarkUp vs Windows Snipping Tool — Side by Side

| Feature | MarkUp | Snipping Tool |
|---------|--------|---------------|
| **Capture** | Annotates live on the page (no capture step) | Screenshot first, then annotate |
| **Context** | Annotations sit ON the website — scrollable, interactive | Static image — loses page context |
| **Text notes** | Numbered badges + formatting (B/I/S/M/L) + multiline | Basic text stamp, no formatting |
| **Drawing** | Freehand brush with color picker + adjustable width | Basic pen, limited colors |
| **Shapes** | Arrows + rectangles (SVG, crisp at any zoom) | Rectangles, arrows, highlighter |
| **Number pins** | Drop numbered markers anywhere (perfect for QA lists) | Not available |
| **Eraser** | Click to undo individual annotations | Erases by stroke |
| **Undo** | Ctrl+Z, unlimited stack | Ctrl+Z, limited |
| **Export** | PNG with path auto-copied — paste directly into Claude Code | PNG to clipboard or file |
| **Output path** | Shows file path in toast — Claude Code reads it directly | Manual file management |
| **Colors** | 5 preset colors (red, blue, green, yellow, white) | Color wheel (more options) |
| **Clipboard** | Auto-copies PNG to clipboard on save | Auto-copies on capture |
| **Dark mode** | Glass morphism dark toolbar | System theme |
| **Works on** | Any webpage (Chrome + Safari extension) | Desktop only (any app) |
| **Delay capture** | Not needed — annotates live | 3s/5s/10s delay options |
| **Screen recording** | Not available | Screen recording built in |
| **Ruler/protractor** | Not available | Ruler tool available |
| **OCR** | Not available | Text extraction (OCR) |
| **Cost** | Free, open source | Free, built into Windows |

## Where MarkUp wins

1. **QA workflow** — annotate directly on the live page, export PNG with path, paste into Claude Code. Zero friction feedback loop.
2. **Numbered pins** — drop 1, 2, 3... markers for ordered issue lists. Snipping Tool can't do this.
3. **Live context** — annotations live on the actual website. You see the real state, not a frozen screenshot.
4. **Text formatting** — bold, italic, 3 sizes. Snipping Tool's text is plain only.
5. **Developer handoff** — the PNG path is auto-copied. Claude Code reads the file directly from the path.

## Where Snipping Tool wins

1. **Screen recording** — MarkUp is annotation-only, no video.
2. **OCR/text extraction** — Snipping Tool can grab text from images.
3. **Works everywhere** — any app, not just browsers.
4. **Ruler tool** — for measuring pixel distances.
5. **More colors** — full color wheel vs 5 presets.

## The killer combo

Use **Snipping Tool** for quick captures and screen recording.
Use **MarkUp** for website QA, design feedback, and developer handoff.
They complement each other perfectly.
