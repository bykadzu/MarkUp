## Full Description

MarkUp lets you draw directly on any live webpage — no screenshot-first workflow, no sign-up. Drop numbered pins, draw arrows, highlight sections, blur sensitive info, add formatted text notes, then export as PNG with one click.

ANNOTATION TOOLS

Pen — Freehand drawing with adjustable width and color. Sketch circles, underline text, mark up anything.

Arrow — Click a start point, click an end point. Clean SVG arrows for pointing out exactly what needs attention.

Rectangle — Drag to draw outlines. Great for boxing UI elements in bug reports.

Highlighter — Semi-transparent color overlay. Highlight text or sections without obscuring them.

Blur / Redact — Backdrop-filter blur that censors passwords, emails, personal data, or anything sensitive before sharing a screenshot.

Text Notes — Numbered badges (1, 2, 3...) with a full formatting toolbar: Bold, Italic, Underline, three sizes (S/M/L). Drag to reposition. Double-click to re-edit. Multiline support — press Enter for new lines, Ctrl+Enter to finalize.

Pin — Drop numbered markers for ordered QA feedback. "Fix #1 here, #2 there."

Stamps — Quick reactions: checkmark, cross, question mark, star. One click to approve, reject, or flag.

Eraser — Click any annotation to remove it. Clean and surgical.

Crop — Region select to save just a portion of the page as PNG.

Undo — Ctrl+Z with an unlimited stack. Undo anything.

9 COLORS + CUSTOM PICKER

Choose from 9 preset colors — red, orange, yellow, green, blue, purple, pink, white, black — or open the custom color picker (rainbow wheel) for any color you want. The active color applies to every tool: pen, arrows, rectangles, text, pins, highlights.

EXPORT OPTIONS

Save as PNG — Downloads the annotated screenshot. The file path is automatically copied to your clipboard so you can paste it directly into Claude Code, Cursor, VS Code, or any AI coding tool.

Copy to Clipboard — Copies the annotated screenshot image to your clipboard. Paste it into Slack, Notion, email, Jira, or anywhere that accepts images.

Export Notes as Markdown — Exports all your text notes and pin labels as a structured markdown file. Perfect for creating actionable feedback lists.

KEYBOARD SHORTCUTS

Ctrl+B — Bold (in text tool)
Ctrl+I — Italic (in text tool)
Ctrl+U — Underline (in text tool)
Ctrl+Z — Undo last annotation
Ctrl+Enter — Finalize text note
ESC — Close MarkUp

BUILT FOR DEVELOPERS AND DESIGNERS

MarkUp was built for the screenshot-and-feedback loop:

1. Click the MarkUp icon
2. Annotate the page — draw, type, pin, blur
3. Hit Save
4. Paste the file path into your dev tool or chat

No more "see attached screenshot" with no context. No more opening a separate app to crop and annotate. No more cropped screenshots that cut off the element you were pointing at. Annotate in context, on the live page.

Use cases:
- Design QA — Mark up what needs fixing, export, hand off
- Bug reports — Circle the broken UI, add numbered notes, paste into the ticket
- Code review context — Screenshot the rendered output, annotate, share with the PR
- Client feedback — Blur sensitive data, add clear callouts
- Documentation — Highlight UI elements for tutorials and guides

PRIVACY

MarkUp does not collect, store, or transmit any user data. Zero analytics. Zero telemetry. Zero cookies. Zero external network requests. No account required.

All annotations live in your browser's memory during the session and are discarded when you close MarkUp. Screenshots are rendered locally using a bundled copy of html2canvas — nothing is uploaded, nothing phones home.

The only permissions requested are activeTab (to overlay annotations on the current page) and scripting (to inject the annotation engine). That's it.

MarkUp is open source: https://github.com/bykadzu/MarkUp

CROSS-BROWSER

Works on Chrome (Manifest V3) and Safari 15.4+ via Web Extension support.

Zero runtime dependencies. Fully offline capable. Fast and lightweight.
