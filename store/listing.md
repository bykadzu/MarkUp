# MarkUp — Draw on Any Website

## Category: Developer Tools

## Short description (132 chars max):
Draw on any website — arrows, pins, blur, text notes — export PNG with file path auto-copied for AI workflows. Zero data collection.

## Detailed description:

MarkUp lets you annotate directly on any live webpage. No screenshot-first workflow. Drop numbered pins, draw arrows, highlight sections, blur sensitive info, then export as PNG with one click.

### 11 Annotation Tools
Pen, Arrow, Rectangle, Highlighter, Blur/Redact, Text (with Bold/Italic/Underline formatting, 3 sizes), Numbered Pins, Stamps (checkmark, cross, question mark, star), Eraser, Crop, and Undo (Ctrl+Z).

### 9 Colors + Custom Picker
Choose from 9 preset colors or pick any custom color. Active color applies to all tools.

### Export Options
- **Save as PNG** — file path auto-copied to clipboard
- **Copy to clipboard** — paste annotated screenshot anywhere
- **Export notes as markdown** — all text notes and pins in one list

### Built for Developer & AI Workflows
The PNG file path is auto-copied on save. Paste it directly into Claude Code, Cursor, VS Code, or any AI coding tool for instant visual context. Perfect for design QA, bug reports, and developer handoff.

### Privacy-First
Zero data collection. No analytics. No tracking. No account required. No external network requests. html2canvas is bundled — works fully offline. Open source.
